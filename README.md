# 📚 Online Book Store SQL Analysis

## 📌 Project Overview

This project analyzes an Online Book Store database using SQL Server. It demonstrates SQL skills by creating a relational database, importing datasets, and solving real-world business problems through SQL queries.

---

## 🎯 Objectives

- Design a relational database
- Import data into SQL Server
- Perform sales analysis
- Analyze customer purchasing behavior
- Track inventory
- Generate business insights using SQL

---

## 🛠️ Tools Used

- SQL Server
- SQL Server Management Studio (SSMS)
